# ネットワーク用語

## ネットワーク

- インターネット: 世界中のコンピュータを接続するグローバルなネットワーク
- LAN (Local Area Network): 限られた地理的範囲内のコンピュータネットワーク
- WAN (Wide Area Network): 広範囲にわたるコンピュータネットワーク
- イントラネット (Intranet): 組織内で使用されるインターネットに繋がっていないネットワーク
- サブネット (Subnet): ネットワークを小さなセグメントに分割すること

## IPアドレス

- IPアドレス (Internet Protocol Address): ネットワーク上のデバイスを識別するための数値アドレス
- グローバルIPアドレス: インターネット上で一意のIPアドレス
- プライベートIPアドレス: ローカルネットワーク内で使用されるIPアドレス
- IPv4: 32ビットのIPアドレス形式 255.255.255.255
- IPv6: 128ビットのIPアドレス形式

## ドメイン

- http://www.example.com
  - `https://` : プロトコル名
  - `www` : サブドメイン(ホスト名)
  - `example.com` : ドメイン名
  - `.com` : トップレベルドメイン (TLD)
  - `www.example.com` : フルドメイン名 (FQDN)
- DNS (Domain Name System): ドメイン名をIPアドレスに変換するシステム

## ネットワークプロトコル

- TCP/IP (Transmission Control Protocol/Internet Protocol): インターネットの基本的な通信プロトコル
- HTTP (Hypertext Transfer Protocol): ウェブ通信のプロトコル
- HTTPS (HTTP Secure): HTTPのセキュアバージョン
- FTP (File Transfer Protocol): ファイル転送のプロトコル
- DHCP (Dynamic Host Configuration Protocol): ネットワーク上のデバイスにIPアドレスを自動的に割り当てるプロトコル
- SSL/TLS (Secure Sockets Layer/Transport Layer Security): データ通信の暗号化プロトコル

## ネットワーク機器

- スイッチ (Switch): ネットワーク内のデバイスを接続し、データを転送するデバイス
- ルーター (Router): ネットワーク間のデータ転送を管理するデバイス

## その他

- ファイアウォール (Firewall): ネットワークセキュリティシステムで、許可されたトラフィックのみを通過させる
- VPN (Virtual Private Network): 公衆ネットワークを通じてプライベートネットワークを構築する技術
- DMZ (Demilitarized Zone): 内部ネットワークと外部ネットワークの間に配置されるセキュリティゾーン
