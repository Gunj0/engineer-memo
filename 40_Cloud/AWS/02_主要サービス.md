# 主なサービス

## サーバー

- EC2 (Elastic Compute Cloud): 仮想マシン
- ECS (Elastic Container Service): コンテナ実行・管理, EC2とFargateの両方で利用可
- EKS (Elastic Kubernetes Service): マネージドKubernetesサービス
- Lightsail: 簡単に使える仮想サーバー・ネットワークのパッケージ
- Batch: バッチ処理

## サーバーレス

- Lambda: イベント駆動型のサーバーレスコンピューティング
- Fargate: 必要に応じてコンテナを実行できるサーバーレスコンテナサービス
- API Gateway: API管理サービス, REST APIやWebSocket APIを提供する
- Step Functions: サーバーレス順次処理ワークフロー

## データベース

- RDB
  - RDS (Relational Database Service): リレーショナルデータベース
  - Aurora: AWS独自の RDB
- NoSQL
  - DynamoDB: キー・バリュー型, 非VPCで利用可能
  - ElastiCache: インメモリデータストア (Redis用とMemcached用がある)
  - DocumentDB: ドキュメント指向データベース (MongoDB互換)
  - Neptune: グラフデータベース
  - Timestream: 時系列データベース
  - QLDB (Quantum Ledger Database): 分散台帳データベース

## ストレージ

- S3 (Simple Storage Service): オブジェクトストレージ
- EBS (Elastic Block Store): ブロックストレージ(EC2用)

## 負荷分散

- ELB (Elastic Load Balancing): 負荷分散サービス
  - ALB (Application Load Balancer): HTTP/HTTPSトラフィック向け
  - NLB (Network Load Balancer): TCPトラフィック向け
  - CLB (Classic Load Balancer): レガシーな負荷分散サービス, 現在はALB/NLBの利用が推奨されている
- Auto Scaling: リソースの自動スケーリングを提供する

## ネットワーク

- VPC (Virtual Private Cloud): 仮想ネットワーク
- CloudFront: コンテンツキャッシュサービス (CDN)
- Route 53: DNSサービス, ドメイン管理, ルーティング
- Transit Gateway: 複数のVPCやオンプレミスネットワークを接続するためのサービス
- Elastic IP: 固定IPアドレスを提供する
- PrivateLink: VPC内でのプライベート接続を提供する
- Direct Connect: 専用線接続サービス, 引き込み工事が必要
  - Direct Connect Gateway: Direct Connectと複数のVPCを接続するためのゲートウェイ
- AWS VPN: オンプレミスネットワークとVPCをVPNで安全に接続するサービス

## メッセージングと通知

- SQS (Simple Queue Service): メッセージキューイングサービス
- SNS (Simple Notification Service): Pub/Sub型のプッシュ通知サービス
- SES (Simple Email Service): メール送信サービス

## メディア

- Elastic Transcoder: メディアファイルの変換サービス
- Elemental MediaLive: ライブビデオストリーミングサービス
- Elemental MediaPackage: 動画配信サービス

## 監視・分析

- CloudWatch: モニタリングと管理サービス、ログをトリガーにしたアクションも可能
- CloudTrail: APIコールのログ記録サービス, 誰が何をしたかを記録する
- Athena: S3データの集計サービス, 必要に応じてサーバーレスでクエリを実行できる
- Redshift: データ集計・分析サービス
  - Redshift Spectrum: S3データ等のクエリサービス, 大量のデータを分析できる

## セキュリティとアイデンティティ

- IAM (Identity and Access Management): ユーザーとアクセス権限の管理を行う, 無料
- Cognito: ユーザー認証と管理を提供するサービス
- GuardDuty: 脅威検出サービス

## CI/CD

- CodeCommit: Gitリポジトリサービス
- CodeBuild: ビルドサービス
- CodeDeploy: デプロイサービス
- CodePipeline: CI/CDパイプラインサービス

## 開発ツール

- Cloud9: クラウドベースの統合開発環境 (IDE)
- Amplify: サーバレス構成を簡単に構築できるOSSフレームワーク
- CloudFormation: インフラストラクチャをコードとして管理するサービス
- Elastic Beanstalk: アプリケーションのデプロイと管理を簡素化するサービス
- OpsWorks: ChefやPuppetを使用した構成管理サービス

## コスト管理

- Billing and Cost Management: コスト分析と予測を提供する
- Budgets: コストと使用量の予算設定とアラートを提供する
