# AWSとは

- Amazon が提供するクラウドコンピューティングサービス
- 2006年にサービスが開始され、世界中で数百種類のサービスを提供している
- Everything as a Service (XaaS)

## 利点

- 可用性: 複数のリージョンや AZ を利用して高い信頼性を実現
- スケーラビリティ: 必要に応じてリソースを迅速に拡張・縮小可能
- コスト効率: 従量課金制で初期投資が不要
- グローバル展開: 世界中のデータセンターを利用可能
- セキュリティ: 多層的なセキュリティ対策とコンプライアンス

## サービスを選ぶ際のポイント

- 可用性: サービスの稼働率や障害対応の体制
- スケーラビリティとパフォーマンス: サービスが負荷に対応できるか
- コスト効率: サービスの料金体系と予算に合うか
- セキュリティ: データ保護やアクセス制御の機能
