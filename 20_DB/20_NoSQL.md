# NoSQL

- Not Only SQL
  - RDBとは異なるデータモデルを採用するデータベースの総称
    - ドキュメント指向
    - キー・バリュー
    - グラフ
    - カラム指向
  - スキーマレスで柔軟なデータ構造
  - メリット
    - 水平スケーリングが容易で、大規模なデータや高いトラフィックに対応できる
    - 柔軟なデータモデルにより、複雑なデータ構造を扱いやすい
  - デメリット
    - トランザクション管理や複雑なクエリが苦手
    - データの整合性を保証するための機能がRDBに比べて弱いことがある

## NoSQLの種類

- Redis
- MongoDB
- Cassandra
- Amazon DynamoDB
