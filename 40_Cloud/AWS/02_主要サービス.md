# 主なサービス

## サーバー

- Amazon EC2 (Elastic Compute Cloud): 仮想サーバー
- Amazon ECS (Elastic Container Service): コンテナ実行・管理
- Amazon EKS (Elastic Kubernetes Service): マネージドKubernetesサービス
- Amazon Lightsail: 簡単に使える仮想サーバー・ネットワーク一式
- AWS Batch: バッチ処理

## サーバーレス

- AWS Lambda: イベント駆動型のサーバーレスコンピューティング
- Amazon API Gateway: APIの作成・管理
- AWS Fargate: サーバーレスコンテナ実行
- AWS Step Functions: サーバーレス順次処理ワークフロー

## データベース

- Amazon RDS (Relational Database Service): リレーショナルデータベース
- Amazon Aurora: AWS独自のRDB
- Amazon DynamoDB: NoSQLデータベース
- Amazon ElastiCache: インメモリデータベース

## ストレージ

- Amazon S3 (Simple Storage Service): オブジェクトストレージ
- Amazon EBS (Elastic Block Store): ブロックストレージ(EC2用)

## ネットワーク

- Amazon VPC (Virtual Private Cloud): 仮想ネットワーク
- Amazon CloudFront: コンテンツキャッシュサービス (CDN)
- Amazon Route 53: DNSサービス、ドメイン管理
- Amazon Direct Connect: 専用線接続サービス
- AWS Transit Gateway: VPC間接続サービス
- Elastic Load Balancing (ELB): 負荷分散サービス
- Elastic IP: 固定IPアドレスを提供する
- AWS PrivateLink: VPC内でのプライベート接続を提供する

## メッセージングと通知

- Amazon SQS (Simple Queue Service): メッセージキューイングサービス
- Amazon SNS (Simple Notification Service): プッシュ通知サービス
- Amazon SES (Simple Email Service): メール送信サービス

## メディア

- Amazon Elastic Transcoder: メディアファイルの変換サービス
- AWS Elemental MediaLive: ライブビデオストリーミングサービス
- AWS Elemental MediaPackage: 動画配信サービス

## 監視・分析

- Amazon CloudWatch: モニタリングと管理サービス
- Amazon Athena: S3データの集計サービス
- Amazon Redshift: データ集計・分析サービス
- AWS Auto Scaling: リソースの自動スケーリングを提供する

## セキュリティとアイデンティティ

- AWS IAM (Identity and Access Management): ユーザーとアクセス権限の管理を行う
- Amazon Cognito: ユーザー認証と管理を提供するサービス
- Amazon GuardDuty: 脅威検出サービス

## 開発ツール

- AWS Cloud9: クラウドベースの統合開発環境 (IDE)
- AWS Amplify: フルスタックのモバイル・ウェブアプリ開発の構築とデプロイ
- AWS CodeCommit: Gitサービス
- AWS CodeDeploy: デプロイサービス
- AWS CodeBuild: ビルド・テストツール
- AWS CodePipeline: ビルド・テスト・デプロイの自動化サービス
- AWS CloudFormation: インフラストラクチャをコードとして管理するサービス
- AWS Elastic Beanstalk: アプリケーションのデプロイと管理を簡素化するサービス

## コスト管理

- AWS Cost Explorer: コスト分析と予測を提供する
- AWS Budgets: コストと使用量の予算設定とアラートを提供する
