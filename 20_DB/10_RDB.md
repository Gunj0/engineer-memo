# RDB

- Relational Database
  - データをテーブル形式で管理する
  - SQL (Structured Query Language) を使用してデータを操作する
  - スキーマを持ち、データの構造を定義する必要がある
  - メリット
    - 複雑なクエリやトランザクションをサポートするため整合性が高い
    - データの正規化により冗長性を減らし、データの整合性を保つことができる
  - デメリット
    - 水平スケーリングが難しいため、大規模なデータや高いトラフィックには適さない

## RDBの種類

- 無償
  - MySQL
  - MariaDB
  - PostgreSQL
  - SQLite
- 有償
  - Oracle Database
  - Microsoft SQL Server
  - Amazon Aurora
