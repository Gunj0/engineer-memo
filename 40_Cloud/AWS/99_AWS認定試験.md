# AWS認定試験

## 種類: https://aws.amazon.com/jp/certification/

### Foundational: 入門レベル

- Cloud Practitioner
  - AWSの基礎知識、クラウドの概念理解
- AI Practitioner
  - AIの基礎、AWSのAIサービスの概要理解

### Associate: 中級レベル

- Solutions Architect: SAA
  - AWSでのシステム設計、アーキテクチャのベストプラクティス理解
- CloudOps Engineer
  - AWS運用監視、トラブル対応、CloudWatch、CloudFormation等の理解
- Developer
  - AWSでのアプリ開発、Lambda、DynamoDB等のサーバレス中心理解
- Data Engineer
  - AWSのデータ分析サービス、ETL、Redshift、Athena等理解
- Machine Learning Engineer
  - AWSの機械学習サービス、SageMaker、MLの理解

### Professional: 上級レベル

- Solutions Architect: SAP
  - 高度なシステム設計、アーキテクチャのベストプラクティス理解
- DevOps Engineer
  - AWSでのDevOps、CI/CD、CloudFormation、CodePipeline等の理解
- Generative AI Developer
  - AWSの生成AIサービス、Bedrock、GenAIの理解

### Specialty: 専門分野

- Advanced Networking
  - AWSのネットワーキングサービス、VPC、Direct Connect等理解
- Security
  - AWSのセキュリティサービス、IAM、KMS、GuardDuty等理解
- Machine Learning
  - AWSの機械学習サービス、SageMaker、MLの高度な理解
