# 主なサービス

## サーバー

- EC2 (Elastic Compute Cloud): 仮想サーバー
- ECS (Elastic Container Service): コンテナ実行・管理
- EKS (Elastic Kubernetes Service): マネージドKubernetesサービス
- Lightsail: 簡単に使える仮想サーバー・ネットワーク一式
- AWS Batch: バッチ処理

## サーバーレス

- AWS Lambda: イベント駆動型のサーバーレスコンピューティング
- API Gateway: APIの作成・管理
- AWS Fargate: サーバーレスコンテナ実行
- AWS Step Functions: サーバーレス順次処理ワークフロー

## データベース

- RDS (Relational Database Service): リレーショナルデータベース
- Aurora: AWS独自のRDB
- DynamoDB: NoSQLデータベース
- ElastiCache: インメモリデータベース

## ストレージ

- S3 (Simple Storage Service): オブジェクトストレージ
- EBS (Elastic Block Store): ブロックストレージ(EC2用)

## ネットワーク

- VPC (Virtual Private Cloud): 仮想ネットワーク
- CloudFront: コンテンツキャッシュサービス (CDN)
- Route 53: DNSサービス、ドメイン管理
- Direct Connect: 専用線接続サービス
- AWS Transit Gateway: VPC間接続サービス
- Elastic Load Balancing (ELB): 負荷分散サービス
- Elastic IP: 固定IPアドレスを提供する
- AWS PrivateLink: VPC内でのプライベート接続を提供する

## メッセージングと通知

- SQS (Simple Queue Service): メッセージキューイングサービス
- SNS (Simple Notification Service): プッシュ通知サービス
- SES (Simple Email Service): メール送信サービス

## メディア

- Elastic Transcoder: メディアファイルの変換サービス
- AWS Elemental MediaLive: ライブビデオストリーミングサービス
- AWS Elemental MediaPackage: 動画配信サービス

## 監視・分析

- Amazon CloudWatch: モニタリングと管理サービス、ログをトリガーにしたアクションも可能
- Amazon CloudTrail: APIコールのログ記録サービス、誰が何をしたかを記録する
- Amazon Athena: S3データの集計サービス
- Amazon Redshift: データ集計・分析サービス
- AWS Auto Scaling: リソースの自動スケーリングを提供する

## セキュリティとアイデンティティ

- AWS IAM (Identity and Access Management): ユーザーとアクセス権限の管理を行う, 無料
- Amazon Cognito: ユーザー認証と管理を提供するサービス
- Amazon GuardDuty: 脅威検出サービス

## 開発ツール

- AWS Cloud9: クラウドベースの統合開発環境 (IDE)
- AWS Amplify: フルスタックのモバイル・ウェブアプリ開発の構築とデプロイ
- AWS CodeCommit: Gitサービス
- AWS CodeDeploy: デプロイサービス
- AWS CodeBuild: ビルド・テストツール
- AWS CodePipeline: ビルド・テスト・デプロイの自動化サービス
- AWS CloudFormation: インフラストラクチャをコードとして管理するサービス
- AWS Elastic Beanstalk: アプリケーションのデプロイと管理を簡素化するサービス

## コスト管理

- AWS Billing and Cost Management: コスト分析と予測を提供する
- AWS Budgets: コストと使用量の予算設定とアラートを提供する
