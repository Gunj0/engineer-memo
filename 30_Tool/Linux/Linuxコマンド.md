# Linux コマンド

## ディレクトリ・ファイル

- ls
  - ファイル一覧表示, -l: 詳細表示, -a: 隠しファイルを含める
- cd
  - ディレクトリ移動
- pwd
  - 現在のディレクトリ表示
- mkdir
  - ディレクトリ作成
- rmdir
  - ディレクトリ削除
- touch
  - ファイル作成
- rm
  - ファイル削除
- cat
  - ファイル内容表示
- grep
  - ファイル検索
- chmod
  - ファイル権限変更

## システムリソース

- top
  - CPU 利用率等のシステムリソース一覧表示
- free
  - メモリ使用率, mac にはない
- ps
  - 起動中のプロセス表示
